Apa itu Statistika?
Statistika adalah ilmu yang mempelajari cara mengumpulkan, menyajikan, menganalisis, dan menginterpretasi data untuk membuat kesimpulan atau keputusan.

Peran Statistika dalam Kehidupan Sehari-Hari:
Menentukan tren pembelajaran siswa
Membantu pengambilan keputusan bisnis (penjualan, stok barang)
Menyusun kebijakan publik berdasarkan survei masyarakat
Memahami kebiasaan konsumsi atau perilaku sosial
Membantu perencanaan kegiatan berdasarkan data partisipasi

Contoh Penggunaan:
Sekolah menggunakan data kehadiran untuk menentukan program motivasi
Pedagang menggunakan data penjualan untuk menentukan produk yang paling laku
Pemerintah menggunakan data kependudukan untuk merancang program bantuan
