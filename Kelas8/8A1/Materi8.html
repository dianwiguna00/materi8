Diagram Panah
Menunjukkan hubungan antara elemen domain dan kodomain dengan panah. Cocok untuk visualisasi relasi dan fungsi sederhana.

Tabel
Menyajikan pasangan nilai dalam bentuk baris dan kolom. Cocok untuk data numerik dan analisis cepat.

Grafik
Mewakili pasangan nilai dalam bidang koordinat (x, y). Cocok untuk melihat pola dan tren fungsi.

Contoh Fungsi:
Misalnya fungsi: f(x) = 2x
Domain: {1, 2, 3}
Kodomain: {2, 4, 6}

Representasi:
Diagram Panah: 1 → 2, 2 → 4, 3 → 6
Tabel: x = 1 f(x) = 2; x = 2 f(x) = 4; x = 3 f(x) = 6
Grafik: Titik-titik: (1,2), (2,4), (3,6)
Contoh Soal 1:
Diberikan fungsi: f(x) = x + 3
Domain: {0, 1, 2, 3}

Pertanyaan:
a. Sajikan dalam bentuk tabel
b. Buat diagram panah
c. Tentukan titik-titik grafik

Jawaban:
Tabel: x = 0 f(x) = 3; x = 1 f(x) = 4; x = 2 f(x) = 5; x = 3 f(x) = 6
Diagram Panah: 0 → 3, 1 → 4, 2 → 5, 3 → 6
Grafik: Titik (0,3), (1,4), (2,5), (3,6)

Contoh Soal 2:
Fungsi: f(x) = 2x - 1
Domain: {-1, 0, 1, 2}

Pertanyaan:
a. Sajikan dalam bentuk tabel
b. Buat diagram panah
c. Tentukan titik-titik grafik

Jawaban:
Tabel: x = -1 f(x) = -3; x = 0 f(x) = -1; x = 1 f(x) = 1; x = 2 f(x) = 3
Diagram Panah: -1 → -3, 0 → -1, 1 → 1, 2 → 3
Grafik: Titik (-1,-3), (0,-1), (1,1), (2,3)
